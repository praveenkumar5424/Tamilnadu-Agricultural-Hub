# Tamil Nadu Agricultural Hub

## Overview

Tamil Nadu Agricultural Hub is a web application that provides useful agricultural information in one place. It is designed to help farmers, students, and anyone interested in agriculture by making important resources easy to access. The application includes information about crops, farming methods, weather updates, and government schemes related to agriculture in Tamil Nadu.

## Features

* View information about different crops.
* Check weather updates to support farming decisions.
* Learn modern and sustainable farming practices.
* Access government schemes and agricultural support programs.
* Simple and user-friendly interface.
* Responsive design that works on both desktop and mobile devices.

## Technologies Used

* HTML
* CSS
* JavaScript
* React.js
* API Integration

## Project Objective

The objective of this project is to provide a single platform where users can access important agricultural information without visiting multiple websites. It aims to make agricultural knowledge more accessible and help users make informed farming decisions.

## Installation

1. Clone the repository.

   ```bash
   git clone https://github.com/praveenkumar5424/Tamilnadu-Agricultural-Hub.git
   ```

2. Navigate to the project directory.

   ```bash
   cd Tamilnadu-Agricultural-Hub
   ```

3. Install the required dependencies.

   ```bash
   npm install
   ```

4. Start the development server.

   ```bash
   npm start
   ```

5. Open your browser and visit:

   ```
   http://localhost:3000
   ```
